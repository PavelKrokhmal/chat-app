# Realtime Chat Application

---

Socket.io, Node.js, and React.js

---

## Installation

```
$ git clone https://github.com/PavelKrokhmal/chat-app
$ cd ./chat-app

For client:

$ cd ./client
$ npm install


For server:

$ cd ./sever
$ npm install

```

## Running

Run server using:

```
$ cd server
$ npm run start
```


Run client using:

```
$ cd client
$ npm run start
```
